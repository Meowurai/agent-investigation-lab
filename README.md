# Agent Investigation Lab

A small experimental lab for building known-ground truth investigation cases for AI data agents.

The goal is to generate synthetic business evidence where the observed data is messy, incomplete, or misleading, while the hidden truth is known.

## Core loop
```text
generate case
→ export observed evidence
→ run reasoner
→ evaluate answer against hidden truth
```

## Cases
### Case 001: Confounded Churn
Support tickets appear related to churn, but the hidden cause is poor
product fit causing both frustration and churn risk.
